# DiaryDecoder

手写日记图片批量转写工具，支持思源笔记自动嵌入。

## 功能

- 扫描指定目录，找出没有对应 `.txt` 的图片
- 调用 [OpenRouter](https://openrouter.ai) 视觉模型识别手写文字
- 逐张审核，批准后保存 `.txt` 并将文字插入思源笔记图片上方
- 支持自动运行批量处理

## 依赖安装

```bash
pip install requests pillow
```

## 使用方法

1. 双击 `diary.bat` 启动
2. 首次运行会弹出「设置」对话框，填写：
   - **图片资产目录**：含手写图片的文件夹路径（可点击「浏览…」选择）
   - **思源 API 地址**：例如 `http://127.0.0.1:6806`
   - **思源 Token**：思源笔记 → 设置 → 关于 → API Token
3. 在顶部输入 OpenRouter API Key，选择模型
4. 点击「开始转写」→ 检查文字 → 点击「✓ 批准 → 保存 + 嵌入思源」

## 快捷键

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+Enter` | 批准并嵌入思源 |
| `Ctrl+S` | 仅保存 txt |
| `Ctrl+→` | 跳过 |
| `Ctrl+←` | 上一张 |

## 配置文件

`config.json` 由程序自动生成和更新，不需要手动创建。
参考 `config.example.json` 了解字段含义。
